# PRACTICA 2  CONCEPTOS BASICOS
1. ¿Qué es una Base De Datos Relacional?
es una base de datosque organiza la informacion en trablas relacionadas entre si.usa logicas normalmente mediante claves para conectar datos de distintas tablas y evitar duplicidad, manteniendo la información consistente,
Cada tabla representa una entidad del mundo real.

2. ¿Qué es una Tabla en el contexto Base De Datos Relacionales?
Es una estructura que almacena datos de forma organizada.
Está compuesta por filas y columnas.
Cada tabla representa una entidad específica.
Ejemplos comunes son clientes, productos o empleados.

3. ¿Qué es una relación en el contexto de Base De Datos Relacionales?
Es la conexión lógica entre dos o más tablas.
Se establece usando claves primarias y foráneas.
Permite vincular información relacionada.
Ayuda a mantener coherencia entre los datos.

4. ¿Qué representa una columna en el contexto de Base De Datos Relacionales?
Representa un atributo de una entidad.
Define el tipo de dato que se almacenará.
Cada columna tiene un nombre específico.
Ejemplos: nombre, precio, fecha o edad.

5. ¿Qué representa un renglón en el contexto de Base De Datos Relacionales?
Representa un registro completo de la tabla.
Contiene valores para cada columna.
Describe una instancia específica de la entidad.
Ejemplo: un cliente individual.

6. ¿Qué es un motor de base de datos relacional?
Es el software que administra la base de datos.
Permite crear, leer, actualizar y eliminar datos.
Ejecuta consultas SQL de manera eficiente.
Garantiza seguridad e integridad de la información.

7. ¿Cuáles son los motores de bases de datos relacionales mas populares?
MySQL es muy usado en aplicaciones web.
PostgreSQL destaca por su robustez y estándares.
Oracle es común en grandes empresas.
SQL Server es ampliamente usado en entornos Microsoft.

8. ¿Cuál es el motor de base datos ligero que no necesita un servidor?
SQLite es un motor de base de datos liviano.
Funciona mediante un solo archivo.
No requiere instalación de servidor.
Es común en aplicaciones móviles y locales

9. ¿Qué significa SQL en el contexto de base datos relacionales?
SQL significa Structured Query Language.
Es el lenguaje estándar para bases de datos relacionales.
Permite consultar y manipular datos.
También se usa para definir estructuras de tablas.

10. ¿Cuáles son los elementos de un sistema de información basado en base de datos relacionales?
Incluye la base de datos y el motor de gestión.
Utiliza aplicaciones que acceden a los datos.
Involucra usuarios y administradores.
También requiere hardware y procedimientos definidos.
